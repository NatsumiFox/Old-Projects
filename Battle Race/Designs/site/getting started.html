Playing the Hack

Thank you for downloading Sonic 3 & Knuckles - Battle Race.

To get started, you'll need an emulator to play the hack. Kega Fusion is great because you can use it to play online with your friends. You can also use Regen and Bizhawk to play the hack with. Unfortunately, Gens and all the related emulators do not work correctly, and therefore the hack is unplayable on them. The hack also works on real hardware using flash carts.

The hack is 2-player only, so you need a friend to play with. Like mentioned, Kega Fusion has a netplay feature, which allows you to play over the internet with someone who lives far away. Other emulators may have this feature as well.

Options

On this screen, you can modify settings regarding the gameplay and additional features. Pressing Up/Down will allow you to navigate through the selections, pressing Left/Right will allow you to change the currently highlighted option, the following options are:

  • SIGNPOST MONITORS – This option decides whether there will be +1 monitors at the end of the level. Enabled by default
  • FLOATING MONITORS – This option decides whether the levels will have additional floating monitors all around them. Enabled by default.
  • MONITOR MODE – This option decides whether the monitor contents of all monitors will contain the default items, or if they will get randomized. Random by default. Some minigames will disable this option.
  • LEVEL BOSSES – This option decides whether to have bosses at the end of the level. Even if you disable them, the level cutscene bosses will still appear. Enabled by default.
  • LAYOUT MODE – Because the layouts are not always great in the original game for Battle Race in mind, some levels are customized to better the racing experience. You can use this setting to choose whether to enable or disable these customizations. Set to altered by default.
  • BOUNCE MODE – This option allows you to enable the Bounce Technology™ created originally by redhotsonic for Sonic Bash. Disabled by default.
  • SECRET OPTION – When enabled, a lot of cool things will happen and you will enjoy the hack infinitely more. Unfortunately, we found out that this option may only be enabled after Half Life 3 is released.

Pressing B will return to the main menu.


Jukebox

On this screen, you can listen to all the music in the hack, and the sound effects from the original game. You can also view the notes being played in real time with the piano display.

Pressing Left/Right will change which music or SFX to play. Once you have highlighted the music or SFX to play, press START button to play it. Pressing Down will highlight the controls menu. Pressing Left/Right while this is highlighted will change the control mode, the following modes are:

  • FADE OUT – Pressing A or C will cause the currently playing music to fade out and then stop.
  • TEMPO – Pressing A will slow down the tempo of the music, pressing C will speed up the tempo of the music.

Pressing Up will highlight the music and SFX selector again. Pressing B will return to the main menu.

Battle Race Level Select

Angel Island Act 1 will be highlighted by default

On this screen, you will be able to choose to play any of the original Sonic 3 & Knuckles levels in the Battle Race mode. Pressing Left/Right will navigate through the level list.

Near the bottom of the screen you can see the number of games each player has won, the left blue side of SCORE is player 1's score, the right side is player 2's score.

Additionally, there is an option for the game to choose a random level. This will avoid repeating levels you've already played. Pressing A, C, or START will select the currently highlighted level. Pressing B will return to the main menu.

This is the random level option, right between DEZ2 and AIZ1 selections!

Mini Game Level Select

Angel Island Act 2 will be highlighted by default

On this screen, you will be able to play any of the mini games introduced in the hack. The screen works similarly to the Battle Race Level Select, except you can only choose one of the following levels:

  • Angel Island Act 2 Mini Game
  • Carnival Night Act 1 Mini Game
  • Launch Base Act 1 Mini Game
  • Sky Sanctuary Mini Game
  • Death Egg Act 2 Mini Game

For more information about the specific mini games, please see the GAMEPLAY page.
